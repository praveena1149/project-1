This project is about global seismic trends.In this project the first step we have to do is data retrieval from usgs website.
After the data retrieval we have to handle the missing values.For handling missing values i used mean method for numeric columns.
After the process of handling missing values we have to store the data in Mysql workbench.
Then we have to gather insights for given queries.
Then the final process is to create a streamlit application to display the queries.
